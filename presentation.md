# Dunder Methods
1. Special methods in python that enable classes to be used in a pythonic way
2. Basics
  * Class instantiation
  ```python
  __new__(cls)
  ```
  * Class initialization
  ```python
  __init__(self, color)
  ```
  * Operators
  ```python
  __add__(x)
  __radd__(x)
  __sub__(x)
  __rsub__(x)
  ```
